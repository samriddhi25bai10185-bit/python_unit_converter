Overview of the Project
The **Python Unit Converter** is a simple, modular, and menu-driven Python application designed to convert **Length**, **Weight**, and **Time** units.  
It follows clean coding practices and a straightforward command-line interface, created as part of the **VIT – Build Your Own Project (BYOP)** evaluation requirements.

The project demonstrates:
- Modular programming  
- Real-world problem solving  
- Accurate mathematical conversions  
- Good documentation and version control  

Features
- Convert **Length** units  
  - Meter ↔ Kilometer ↔ Centimeter  
- Convert **Weight** units  
  - Kilogram ↔ Gram ↔ Pound  
- Convert **Time** units  
  - Second ↔ Minute ↔ Hour  
- Simple menu-based navigation  
- Accurate conversion formulas  
- Clean and modular code  
- Easy to extend with more units  

 Technologies / Tools Used
- **Python 3.x**  
- Standard Python libraries  
- **Git & GitHub** for version control  
- CLI (Command Line Interface)
